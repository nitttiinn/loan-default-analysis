# 🏦 Loan Default Analysis

## Project Overview
Analysis of 148,670 loan records to identify key factors 
that lead to loan defaults.

## Key Findings
- Overall default rate: 24.64%
- LTV is the strongest predictor of default
- Very High LTV loans have 65% default rate
- Very Large loans have 48.88% default rate
- North-East region has highest default rate (30.4%)
